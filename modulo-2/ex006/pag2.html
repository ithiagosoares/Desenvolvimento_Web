Grouping Tags e Semantic Tags

  • Header
    • Áreas relativas a cabeçalhos(Podem conter menus de navagação).
  • Nav
    • Define uma área que possui os links de navegação pela estrutura de páginas do website (Pode estar dentro de um Header).
  • Main
    • É um agrupador usado para delimitar o conteúdo principal do nosso site (Seções, artigos e conteúdos periféricos).
  • Section
    • Cria seções para sua página ("uma seção é um agrupamento temático de conteúdos, tipicamente com um cabeçalho").
  • Article
    • É o elemento que vai conter o conteúdo de forma idependente (Pode ser lido independente).
  • Aside
    • Delimita um conteúdo complementar ao conteúdo principal de um artigo.
  • Footer
    • Cria um rodapé para o site (Não faz parte do conteúdo, mas possui algumas informações sobre o site ou outra coisa).
