Shorthands

As Shorthands são atalhos para utilizar as formatações de fontes:

            
                font-family: Arial, Helvetica, 
                font-size: 1em;
                sans-serif;                
                font-style: italic;
                font-weight: bold;

Simplificanco: Para simplificar essas linha, pode ser usao a shorthand "font": 

"font: italic bold 1em Arial, Helvetica, sans-serif"

Para fazer essa simplificação é preciso seguir uma ordem:

                    
                font-style
                font-variant
                font-weight
                font-size/line-height
                font-family
