Tags:

Itálico e Negrito:

Caso necessário a escrita em Itálico e em Negrito,
é preciso o uso de duas tags, como < em > e < strong >.


Marca texto:

Agora, caso você queira marcar um trecho importante>,
é necessário outras tags, < mark > e < /mark >.

Texto pequeno:

A tag < small > é utilizada para diminuir o tamanho de um texto.>

Texto deletado:

A tag < del > é utilizada para marcar um texto com um
risco no meio dele.

Texto inserido:

sublinhar um texto é necessário o uso da tag < ins >

Textos Sobrescritos2 e Subscritos2

Caso necessário o uso de elevar um número como: x2,
ou escrever H2O, é preciso as tags < sup > e < sub >.

Citaões:

Para colocar algum tipo de trecho citado com as aspas, basta usar a tag < q >.

Para citações mais longas, pode-se usar um outro tipo de tag: < blockquote >.

Abreviações:

No caso de abreviações de sites como: HTML (Hyper Text Markup Language),
é necessário o uso das tags < abbr titlhe > (alguma coisa) </abbr >
