Formato da lista:

Listas Ordenadas (Ordered Lists):

Para formar uma lista ordenada, é necessário duas tags: < ol > < /ol > e < li > < /li >

  2. Item 2
  3. Item 3

Listas não Ordenas (Unordered Lists):

Para formar uma lista ordenada, é necessário duas tags: < ul > < /ul > e < li > < /li >

  • Item 1
  • Item 2
  • Item 3

Lista de Definições:

Nas listas de definições, são utilizadas 3 tags: < dl > e < /dl >, < dt > e < /dt >, < dd > e < /dd >
