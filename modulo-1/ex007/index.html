Vamos criar um link:

Existe um atributo chamado target, que apresenta como os links serão abertos para você:


    _blank
    Vai abrir o link em uma nova janela em branco.
    _self
    Vai abrir o link na janela ou frame atual.
    _top
    Vai desfazer todos os frames e abrir o destino no navegador completo.
    _parent
    Similar ao uso do _top em uma referência à janela mão.
    nome-do-frame
    Caso esteja usando frames, indicar o nome da janela a abrir.

Esta tag permite indicar qual é o idioma do site que está em um link, utilizando a tag herflang="pt-br", por exemplo.

Por exemplo: 

<a>href="URL" hreflang="pt-br"></a>

Indicar qual é a natureza do link, utilizando o atributo rel:

    next
    Indica que o link é para a próxima parte do documento atual.
    prev
    Indica que o link é para a parte anterior do documento atual.
    author
    Indica que é um link para o site do autor do artigo atual.
    external
    Indica que é um link para outro site qeu não faz parte do site.
